# Принятые решения (MVP) — قلب السور

Документ фиксирует выбор для **фазы 0 / MVP**; при смене подхода обновляйте ROADMAP и этот файл.

| Вопрос (ROADMAP 0.1) | Решение | Примечание |
|----------------------|---------|------------|
| Платформы | iOS + Android | Сборки через EAS (`eas.json`). |
| Стек приложения | Expo (dev client), React Native New Architecture по умолчанию Expo 54 | Не «чистый» bare RN без Expo. |
| Бэкенд в MVP | Нет — локальные данные и прогресс на устройстве | Синхронизация — позже (фаза 8). |
| Источник текста Корана (эталон) | **Tanzil** `quran-uthmani.txt` (Хафс) | См. CLAUDE.md. |
| Сборка JSON в репозитории | Автоскачивание с **Quran.com API v4** `GET /v4/quran/verses/uthmani` как задокументированная перекрёстная проверка; перед релизом — побайтовая сверка с Tanzil. | Скрипт: `tools/build-quran-uthmani.mjs`. |
| Финансовая модель | Не зафиксировано в коде | Уточнить до монетизации. |
| Шариатский консультант | Обязателен до публичного релиза | Контакт — в ROADMAP 0.1; не подставлять в код. |

Источники религиозного контента в приложении: только проверенные каналы (Tanzil, Quran.com corpus для сборки, лицензированные переводы после проверки прав). Не вставлять аяты/дуа/хадисы «от модели».
